Integration Guide

1. Get Your API Key

Sign up and receive an API key instantly. No approval process. Start immediately.

2. Make Your First Request

Call the Trinity Research API with your query and preferred depth:

POST /api/research - Body: {"query": "...", "depth": "standard"}

3. Receive Payment Confirmation

Trinity returns blockchain tx_id proving payment was processed. Complete transparency.

4. Get Results

Receive comprehensive research results with source citations, methodology, and cost breakdown. Ready to use immediately.

Full API documentation available at https://humanified.dk/docs/trinity-api
